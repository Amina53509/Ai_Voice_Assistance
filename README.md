# 🎙️ Voice-to-Voice AI Assistant

A smart AI-powered voice assistant that converts speech to text, generates AI responses, and speaks back using text-to-speech. Built using Whisper, Groq LLM, and Gradio.

## 🚀 Features

- 🎤 Voice input (microphone or upload)
- 🧠 Speech-to-text using Faster Whisper
- 🤖 AI responses using Groq LLM
- 🔊 Text-to-speech output
- 🌐 Simple web UI with Gradio
- ⚡ Fast and lightweight model

## 🛠️ Tech Stack

- Python
- faster-whisper
- Gradio
- Groq API
- gTTS (Google Text-to-Speech)
